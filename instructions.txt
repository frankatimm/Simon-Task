Start instructions:

General instructions:
Before we get started we will provide you with some short instructions and explain what your
task is going to be.
You will be presented a grey, blue or green circle.
Your task is press "B" if there is a grey circle, "J" if there is a blue circle and "F" if there is a green circle.
Your answers should ensue as fast and accurate as possible.
In order to prepare you for the experimental setup you will first run through some practice trials before you proceed to the main trials.
Before each task you will see what exactly to do and which key you have to press.
If you feel ready to start the practice trials, please press enter.
